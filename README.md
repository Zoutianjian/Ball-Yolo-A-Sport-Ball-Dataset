# Ball-Yolo-A-Sport-Ball-Dataset
This is a high-quality video-based sport ball dataset, including tennis, table tennis, and football, with a total of over thirty thousand instances.

Our dataset is currently in a pre-release status (informally announced), primarily to facilitate search and download. It is intended solely for research and academic purposes (the corresponding paper is currently under review). In the future, we will provide a comprehensive update and release the dataset on a professional dataset repository.

For official verision or reference please refer to https://doi.org/10.5061/dryad.3bk3j9m13
